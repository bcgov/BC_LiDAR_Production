*These files are used in the LASTools las2las tool to properly format the coordinate reference system in the header of every LiDAR file and should NOT be deleted, moved, or edited without prior permission.

Each text file contains a single string which represents the WKT compound CRS for each UTM zone in BC for NAD83(CSRS) with a vertical datum of CGVD2013.

Vertical Datum

CGVD2013 = EPSG 6647


Horizontal Datums

NAD83(CSRS) / UTM zone 7N = EPSG 3154
NAD83(CSRS) / UTM zone 8N = EPSG 3155
NAD83(CSRS) / UTM zone 9N = EPSG 3156
NAD83(CSRS) / UTM zone 10N = EPSG 3157
NAD83(CSRS) / UTM zone 11N = EPSG 2955


Compound Projections

NAD83(CSRS) / UTM zone 7N + CGVD2013 height = EPSG 6650
NAD83(CSRS) / UTM zone 8N + CGVD2013 height = EPSG 6651
NAD83(CSRS) / UTM zone 9N + CGVD2013 height = EPSG 6652
NAD83(CSRS) / UTM zone 10N + CGVD2013 height = EPSG 6653
NAD83(CSRS) / UTM zone 11N + CGVD2013 height = EPSG 6654
