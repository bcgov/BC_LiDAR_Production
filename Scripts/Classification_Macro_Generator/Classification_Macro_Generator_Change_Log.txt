Classification_Macro_Generator_v4.0 - January 13th, 2026
- added a second workflow to copy LAZ tiles from a source location to a destination, then convert + generate macros
- added safe copy behavior (copy to .part then rename) and an option to overwrite or skip existing files
- added download progress + speed reporting (percent, files copied, MiB/s) in the GUI
- added per-tile conversion with a user-selected max conversions setting (parallel conversion)
- updated file organization so Urban vs Regular matching uses tile stems (more reliable) and the app reports both output folder paths consistently
- suppressed LAStools pop-up windows during conversion (stdout/stderr captured and printed to the app console)
- improved packaged-build console behavior (guards missing stdout/stderr; tqdm only disabled when no console streams exist)
- redesigned the GUI using ttk styling with a clearer two-column layout for the two workflows
