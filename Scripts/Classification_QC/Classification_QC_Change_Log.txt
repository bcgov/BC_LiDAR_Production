Classification_QC_v2.7 - February 27th, 2026 (update by Nikolay Senilov

- Window title now uses VERSION dynamically
- Added small gray version label to bottom of GUI
- GUI: Replaced plain tkinter layout with ttkbootstrap (cosmo theme).
- Two-column panel separating Main QC and Optional QC rasters, round-toggle checkboxes, auto-sized window height, new QC_icon.ico applied everywhere.

- Robustness: Full try/except around processing pipeline, os.system return code checks for lasgrid64/blast2dem64, .las-only directory validation.
- Spec: Updated to collect_all('ttkbootstrap') and bundle QC_icon.ico.
